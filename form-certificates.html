Add or delete certificates. Changes sync to all devices instantly.

One-time setup needed to sync across devices

  1. Go to jsonbin.io → sign up free
  2. Click + Create Bin → paste [] as content → Save
  3. Copy the BIN ID (looks like: 66a1b2c3d4e5f6...)
  4. Go to API Keys tab → copy your X-Master-Key
  5. Open this file in a text editor → find JSONBIN_ID and JSONBIN_KEY at the top of the <script> and paste your values
